# Simple-Chinese-CV-Template
一个简洁的中文简历模板，
没什么特别的做这个玩意的原因，市面上能找到的中文简历对我来说都太繁琐了，所以自己弄了一个。
如果真有人能看到这玩意，对于不懂的人讲讲使用方法吧。
没什么复杂的，就点开main.tex这个玩意，把里面的东西一复制，到Overleaf里创建新项目粘贴进去。
或者你乐意的话直接下载再Upload上去。
然后，点开左上角的File - Settings - Compiler - Compiler - XeLaTeX。
最后点一下Recompile，看看有没有什么问题，没问题下载PDF就完事了。
如果想看一下效果的可以瞅一眼Sample ^_^
